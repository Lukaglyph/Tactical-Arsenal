TACTICAL ARSENAL - README『TACTICAL ARSENAL』は、将棋をベースとした伝統的なボードゲームのルールに、サイバーパンクな世界観と近代的な兵装（アーセナル）カスタマイズを融合させた、次世代のタクティカル・ボードゲームです。1. ゲーム概要プレイヤーは自軍のユニットを操作し、敵軍の「コア（K）」を撃破することを目指します。対人戦（PvP）だけでなく、将棋の思考アルゴリズムを応用した戦略強化型AIとの対戦が可能です。2. ユニット紹介各ユニットは独自の移動パターンを持ち、敵陣（奥の3段）に入ると「成り（プロモーション）」が可能です。ユニット記号特徴コアK王将に相当。この駒を失うと敗北。全方向に1マス移動。ストライカーP歩兵に相当。前方1マスに移動。ピアッサーL香車に相当。前方へどこまでも直進可能。ジャンパーN桂馬に相当。前方の斜め2マス先へ飛び越えて移動。ガードS銀将に相当。前方3マスと斜め後方2マスに移動。ベテランG金将に相当。周囲6マス（真後ろ以外）に移動。スナイパーB角行に相当。斜め方向にどこまでも移動可能。バスターR飛車に相当。縦横方向にどこまでも移動可能。3. 兵装（アーセナル）システムゲーム開始前に、各ユニットに「追加兵装」を装備させることができます。合計コストは12以内に収める必要があります。標準兵装 (Cost 0): 追加機能なし。シールド (Cost 1): 一度だけ敵の攻撃を無効化し、駒が取られるのを防ぎます。EMP (Cost 2): 着地時、周囲8マスの敵を一時的に「麻痺（STUN）」させ、移動不能にします。スナイプ (Cost 2): ストライカー（P）専用。前方の射程が+1されます。ブースター (Cost 3): 移動後、もう一度連続で行動可能です（B, Rは装備不可）。グラビティ (Cost 3): 着地時、周囲の敵を1マス外側に弾き飛ばします。4. 戦略強化型AIについて本バージョンのAIは将棋の評価関数をベースに強化されています。駒の価値評価: 駒の種類に応じたスコア計算を行い、有利な交換を狙います。前進思考: 将棋のセオリーに基づき、中央の支配や敵陣への前進を優先します。勝利優先: 詰みの状態（コアを取れる状況）を逃さず、最短ルートで勝利を狙います。5. 基本操作モード選択: 「対人戦」または「CPU戦」を選択します。カスタマイズ: ユニットを選び、好みの兵装を装備します。対局開始:自分の駒をタップすると移動可能なマスがハイライトされます。持ち駒がある場合は、自分の手番で空いているマスに配置できます。敵の駒を取ると自分の持ち駒になります（兵装はリセットされます）。6. 勝敗条件勝利: 敵の「コア（K）」をキャプチャする。敗北: 自軍の「コア（K）」をキャプチャされる。
