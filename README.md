TACTICAL ARSENAL (タクティカル・アーセナル)『TACTICAL ARSENAL』は、将棋の伝統的な戦略性と、サイバーパンクな兵装（アーセナル）カスタマイズを融合させた、ブラウザで遊べる次世代タクティカル・ボードゲームです。1. ストーリー / コンセプトプレイヤーは戦術指揮官となり、エネルギーコストの制限内でユニット（駒）をカスタマイズし、敵軍の「コア（K）」を撃破するための作戦を立案します。将棋のアルゴリズムに基づいた戦略思考型AIが、あなたの前に立ちはだかります。2. ユニットと移動ルール各ユニットは、敵陣（奥の3段）に入ると「成り（プロモーション）」が可能で、移動性能が強化されます。ユニット記号移動パターンコアK全方向に1マス移動（王将）。ストライカーP前方に1マス移動（歩兵）。ピアッサーL前方にどこまでも進める（香車）。ジャンパーN前方斜め2マス先へ飛び越える（桂馬）。ガードS前方3マス＋斜め後方2マスへ移動（銀将）。ベテランG真後ろ以外の周囲6マスへ移動（金将）。スナイパーB斜め方向にどこまでも進める（角行）。バスターR縦横方向にどこまでも進める（飛車）。3. 兵装（アーセナル）カスタマイズゲーム開始時に、各ユニットに特殊な兵装を付与できます。合計コストは 12 以内に制限されています。標準兵装 (Cost 0): 追加機能なし。シールド (Cost 1): 一度だけ敵の攻撃を無効化します。EMP (Cost 2): 移動した際、周囲8マスの敵を1ターンの間麻痺（移動不可）させます。スナイプ (Cost 2): ストライカー(P)専用。前方の射程が2マスになります。ブースター (Cost 3): 移動後にもう一度連続で行動できます。（※B, Rは装備不可）グラビティ (Cost 3): 移動した際、隣接する敵を1マス外側に弾き飛ばします。4. 強化版AI (Strategic AI v2.0)本バージョンでは、将棋の評価関数を参考にした高度なAIが搭載されています。評価関数: 駒の価値、成りやすさ、中央の支配度、コアへの距離を計算して最適な一手を選択します。勝利への執念: 王手（コアへの攻撃）が可能な状況では、確実に勝利を狙いに行きます。5. 遊び方モード選択:「対人戦」：1つの画面で2人で交互に操作します。「CPU戦」：戦略強化型AIと対戦します。カスタマイズ:画面左のユニットリストからユニットを選択。画面右のモジュールリストから兵装を選択。画面上部のコストゲージを確認し、コスト12以内に収めて「作戦開始」。対局:自分の駒をタップして移動先を選択します。敵の駒を取ると自分の持ち駒となり、好きな時に盤面の空きマスへ配置できます。敵の「コア」を先に取ったプレイヤーの勝利となります。6. 動作環境モダンブラウザ（Chrome, Edge, Safari, Firefoxなど）スマートフォン / タブレット対応（レスポンシブ設計）開発・ライセンスEngine: Pure JavaScript / Tailwind CSSIcons: Lucide IconsVersion: 2.1.0 (Strategic AI & Visual Effect Update)
