サイバーパンクな世界観を舞台にした、アーセナル（兵装）カスタマイズ機能付きの戦術ボードゲームです。将棋のような戦略性に、ユニット固有の特殊能力とプレイヤーによる装備の組み合わせが加わり、高度な駆け引きを楽しめます。

## 🚀 特徴

- **アーセナル・カスタマイズ**: 全8種類の兵装を各ユニットに自由に装備。限られたエネルギー容量（コスト）内で最強の布陣を構築せよ。
- **ステルス・システム**: 敵から捕捉（捕獲）されない特殊状態。EMP攻撃以外では突破不可能な隠密戦術が可能。
- **EMP & 重力兵器**: 周囲を麻痺させるEMP、敵を弾き飛ばすグラビティなど、盤面をコントロールする特殊能力。
- **対AI戦 & 対人戦**: 一人でじっくり楽しめる Ultimate AI モードと、デバイス一台で対戦できるPVPモードを搭載。
- **サイバーパンクUI**: ネオンカラーを基調とした、直感的かつスタイリッシュな操作画面。

## 🛠 8種類のアーセナル（兵装）

| 名称 | 効果説明 |
| :--- | :--- |
| **標準兵装** | 追加機能なし。コスト0。 |
| **シールド** | 一度だけ敵の攻撃を完全に無効化する。 |
| **EMP** | 着地時、周囲8マスの敵を1ターン麻痺させる。 |
| **ブースター** | 移動後にもう一度行動可能。 |
| **スナイプ** | ストライカー（歩兵）の射程を前方に+1強化。 |
| **グラビティ** | 着地時、周囲の敵を1マス外側に弾き飛ばす。 |
| **リペア** | 着地時、周囲の味方の状態異常を即座に回復。 |
| **ステルス** | 移動後、次の自ターン開始まで敵の捕獲対象外となる。 |

## 🕹 遊び方

1. **カスタマイズフェーズ**: 
   各駒に兵装を割り当てます。画面上部のエネルギー容量（12）を超えないように注意してください。
2. **デプロイ**: 
   「Deploy Units」でゲーム開始。
3. **ゲームルール**: 
   - 相手の「コア（K）」を先に取ったプレイヤーの勝利です。
   - 敵の陣地に進入するとユニットが昇級し、シールドが自動展開されます。
   - 捕獲した駒は自分の持ち駒として再配置可能です（再配置時は標準兵装になります）。
